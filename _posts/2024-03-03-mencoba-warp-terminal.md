sebagai pengguna linux, menggunakan terminal sudah menjadi keseharian. terutama ketika install aplikasi maupun konfigurasi settingan tertentu.
ada banyak aplikasi terminal yang tersedia di linux, di beberapa distro linux bahkan menjadi default bawaan.  
saya sendiri di laptop menggunakan terminal dari [kitty](https://sw.kovidgoyal.net/kitty/) alasan memilih kitty untuk menjadi default terminal harian saya karna kitty ini simple dan juga kustomisasi nya banyak.

nah kali ini saya akan mereview salah satu aplikasi terminal yang memiliki fitur cukup menarik yaitu Warp Terminal.

# Warp Terminal

Warp terminal adalah aplikasi terminal berbasis rust yang dibuat untuk memudahkan pengguna yang belum terbiasa menggunakan terminal.
dengan tampilan yang sangat user friendly menggunakan warp ini tidak akan sesulit menggunakan terminal biasanya.

# Fitur Warp Terminal

Fitur unggulan yang dihadirkan di Warp terminal :

- Warp Drive
- Warp AI

# Warp Drive

dengan fitur ini kita bisa menyimpan beberapa command dan membagikannya ke tim yang tergabung ke dalam tim kita.
fitur ini cukup bagus karna beberapa command yang kita simpan bisa kita kategorikan ke beberapa folder, sehingga ini
lebih memudahkan untuk penggunaan nantinya.

nah ini adalah fitur yang tidak ada di terminal lain, yaitu AI. dengan AI yang langsung terintegrasi
dengan terminal nya langsung. kita bisa lebih terbantu saat akan menjalankan command tertentu di terminal.

# Ganti Tema

di warp terminal untuk ganti tema cukup mudah, kalian hanya perlu pergi ke menu > setting > appearance
lalu kalian klik di bagian tema. setelah itu ganti dengan tema yang ingin kalian gunakan

saya sendiri disini coba menggunakan tema solarized dark.

# Kelebihan dan Kekurangan Warp Terminal

**Kelebihan :**

- terintegrasi dengan AI
- tampilan user friendly, terutama untuk user yang jarang menggunakan terminal.
- pilihan tema banyak dan bisa custom.
- performa cepat, karena terminal ini dibuild dengan Rust.
- fitur workspace untuk menyimpan command penting dan bisa dibagikan ke pengguna yang tergabung di tim.

**Kekurangan :**

- pertama kali install butuh koneksi internet untuk login user.
- berbayar
- fitur di versi gratis terbatas.
- fitur AI hanya terbatas untuk 20 perintah perhari nya untuk satu user.
- fitur AI hanya berfungsi ketika terkoneksi dengan internet.
- versi gratis maksimal user yang tergabung dalam 1 tim hanya 5 orang.
